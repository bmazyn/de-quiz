# Scripts

This folder contains one-off manual utility scripts for data manipulation and maintenance.

## Usage

These scripts are **not used at runtime** and are not part of the build process.

Run manually when needed:

```bash
node scripts/<script-name>.js
```

## Available Scripts

- `add-foundation2.js` - Add Foundation 2 data
- `convert-to-csv.js` - Convert JSON quiz data to CSV format
- `filter-cards.js` - Filter quiz cards by criteria
- `generate-csv.js` - Generate CSV from quiz data
- `remove-fnd-prefix.js` - Remove foundation prefix from card IDs
- `rename-ids.js` - Rename card IDs in bulk
